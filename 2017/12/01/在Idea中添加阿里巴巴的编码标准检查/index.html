在Idea中添加阿里巴巴的编码标准检查

在公司大牛的推荐下，今天接触了一个idea的插件：“Alibaba Java Coding Guidelines”。插件可以按照阿里巴巴的JAVA编码规范来帮助检查代码，并给出改进建议。

安装

在Idea->File->Setting->Plugins->Browse repositories 搜索 Alibaba 安装重启即可。

使用示例

重启idea后，插件会自动对代码进行分析，并对不同的错误进行提示。也可使用鼠标右键进行代码的检查。检查完成后插件会将不符合规约的代码按Blocker/Critical/Major三个等级显示在下方，并在左侧相关规范问题，右侧是问题与修复建议代码。
